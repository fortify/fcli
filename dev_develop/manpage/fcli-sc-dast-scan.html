Name

fcli-sc-dast-scan - Manage ScanCentral DAST scans and results.

Synopsis

fcli sc-dast scan [-h] [COMMAND]

Description

Options

-h, --help

Show this help message and exit. Use 'fcli <command> -h' to display help for subcommands.

Commands

complete

Complete a scan.

delete, rm

Delete a scan.

download

Download scan fpr, logs, results, settings or site tree.

get

Get the details for a single scan.

import-findings

Import scan findings.

list, ls

List and query scans.

pause

Pause a scan.

publish

Publish scan results.

resume

Resume a scan

start

Start a new scan.

wait-for

Wait for one or more scans to reach or exit specified scan statuses.
