Name

fcli-ssc-artifact - Manage SSC artifacts.

Synopsis

fcli ssc artifact [-h] [COMMAND]

Description

Options

-h, --help

Show this help message and exit. Use 'fcli <command> -h' to display help for subcommands.

Commands

approve

Approve an artifact.

delete, rm

Delete an artifact.

download

Download an artifact.

get

Get artifact details.

import-debricked

Import results from Debricked as SSC application version artifact.

list, ls

List application version artifacts.

purge

Purge an artifact.

upload

Upload an artifact.

wait-for

Wait for SSC artifact to reach or exit specified artifact statuses.
