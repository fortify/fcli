Name

fcli-util - Various fcli utility commands that don't fit in any of the other modules

Synopsis

fcli util [[-h] [--env-prefix=<envPrefix>] [--log-file=<logFile>] [--log-level=<logLevel>]] [COMMAND]

Description

Generic fcli options:

-h, --help

Show this help message and exit. Use 'fcli <command> -h' to display help for subcommands.

--env-prefix=<envPrefix>

Environment variable prefix for resolving default option and parameter values. Default value is FCLI_DEFAULT.

--log-file=<logFile>

File where logging data will be written. If not specified, no logging data will be written.

--log-level=<logLevel>

Set logging level. Note that DEBUG and TRACE levels may result in sensitive data being written to the log file. Allowed values: TRACE, DEBUG, INFO, WARN, ERROR

Commands

all-commands

Commands for listing information about all fcli commands

auto-complete

Commands for generating fcli auto-completion scripts
